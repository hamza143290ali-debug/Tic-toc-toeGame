# 🎮 Tic-Tac-Toe Game

## 📌 Overview

This project is a simple and interactive **Tic-Tac-Toe game** built using **HTML, CSS, and JavaScript**. It allows two players to play the game in turns (Player X and Player O) with sound effects and a clean user interface.

The purpose of this project is to practice core front-end development concepts and improve logical thinking using JavaScript.

---

## 🚀 Features

* 🟢 Two-player gameplay (X vs O)
* 🔊 Sound effects on each click and button press
* 🔄 New Game reset functionality
* 🧠 Automatic winner detection
* 🚫 Prevents clicking on already selected boxes
* 🎯 Clean and responsive UI using Grid and Flexbox

---

## 🛠️ Technologies Used

* **HTML5** → Structure of the game
* **CSS3** → Styling, layout (Grid & Flexbox)
* **JavaScript (ES6)** → Game logic and interactivity

---

## 📚 Concepts Implemented

This project demonstrates the use of the following programming concepts:

* **DOM Manipulation** → Accessing and updating HTML elements
* **Event Handling** → Handling user clicks
* **Arrays** → Storing winning patterns
* **Loops (forEach)** → Iterating over buttons
* **Objects (Audio API)** → Handling sound effects
* **Conditional Logic** → Checking winner conditions
* **Functions** → Resetting game and checking winner

---

## 🎮 How to Play

1. Open the game in your browser
2. Player **O** starts first
3. Players take turns clicking on empty boxes
4. The game automatically detects the winner
5. Click the **New Game** button to restart

---

## 🔊 Sound Effects

* Click sound plays when a box is selected
* Button sound plays when the "New Game" button is clicked
